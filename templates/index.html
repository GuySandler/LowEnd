Frequently Asked Questions

What is LowEnd?

LowEnd is a YSWS (You Ship, We Ship) program that challanges participants to learn and create low-level programs to use as little resources (CPU, RAM, Storage) as possible.

What can I make?

Any program that fits the current theme and meets the rquirements

What are the requirements?
  • The program must be able to run
  • Spend at least 2 Hacktime hours making it (disclaimer and download link here)
  • Have at least X of the features from the required feature list of the current theme.
  • 1 submission per theme

Do I have to do every theme?

No, you can choose which themes to participate in, but the more you do the more points you get!
